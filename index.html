Properties for CSS

Text propertises reference

Property Syntax Descriptoin
direction direction: ltr | rtl | initial | inherit Define the text direction/writing direction.
text-align text-align: left | right | center | justify | initial | inherit Sets the horizontal alignment of the inline content like text and images in its parent block element such as heading or paragraph.
text-decoration text-decoration: none | [ underline | overline | line-through | blink ] one or more values | initial | inherit Specifies the decorations to be applied on the text content of an element such as underline, overline, etc.
line-height line-height: length | percentage | number | normal | initial | inherit Sets the height (also called leading) between lines of text in a block-level element such as a paragraph.

Visual propertises reference

Property Syntax Descriptoin
display display: inline | block | contents | flex | flow | flow-root | grid | inline-block | inline-flex | inline-grid | inline-table | list-item | run-in | table | table-caption | table-column-group | table-header-group | table-footer-group | table-row-group | table-cell | table-column | table-row | none | initial | inherit Specifies the type of box generated by an element.
z-index z-index: integer | auto | initial | inherit Specifies the layering or stacking order for the positioned elements i.e. elements whose position value is one of absolute, fixed, or relative.
overflow overflow: auto | hidden | scroll | visible | initial | inherit Specifies whether to clip content, render scroll bars or display overflow content when content overflows the element's content area.
